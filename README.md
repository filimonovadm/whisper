# Транскрибация аудиофайлов с помощью Whisper

## Установка

Для использования этого инструмента вам потребуется установленная утилита `whisper`. Если она у вас не установлена, вы можете установить ее, следуя официальной инструкции.

## Использование

Для транскрибации аудиофайла выполните следующую команду в терминале:

```bash
whisper <имя_файла> --output_dir .
```

Где `<имя_файла>` - это путь к вашему аудиофайлу (например, `лекция-1.m4a`).

Флаг `--output_dir .` указывает, что файлы с транскрибацией (в форматах .txt, .srt, .vtt, .tsv, .json) будут сохранены в текущей директории.

### Пример

```bash
whisper лекция-1.m4a --output_dir .
```

## Использование LLM для суммаризации транскрибации

После получения текстового файла (`.txt`) с транскрибацией, вы можете использовать Large Language Model (LLM) для создания краткого обзора или извлечения ключевых моментов.

Пример промпта для суммаризации:

```
Проанализируй следующий текст, который является транскрипцией лекции. Создай краткое саммари, выдели основные темы и ключевые выводы.

[Вставьте сюда содержимое файла .txt]
```
